🚀 Rust 区块链项目 - Windows 可执行版本
=============================================

📅 构建日期: 2025年10月23日
📦 版本: v1.0.0 (包含完整P2P同步功能)

🎯 项目简介
这是一个用 Rust 语言实现的完整区块链技术演示平台，包含：
- 完整的区块链系统 (PoW/PoS/DPoS共识算法)
- 数字签名和钱包管理系统
- Web区块链浏览器
- P2P分布式网络和节点同步
- Solana智能合约集成

✅ 功能特性
- 区块创建、验证和区块链管理
- Ed25519数字签名交易系统
- 多重共识算法支持
- 完整的P2P节点同步机制
- Web界面区块链浏览器
- 数据持久化存储

📁 包内容
- blockchain_project.exe - 主程序可执行文件
- README.md - 详细使用文档
- readme-p2p-sync.md - P2P同步功能详细说明
- blockchain.json - 区块链数据文件

🚀 快速开始
1. 双击运行 blockchain_project.exe
2. 按照菜单提示进行操作
3. 选择 8 进入 P2P 网络操作
4. 选择 6 进行区块链同步

🔧 系统要求
- Windows 10 或更高版本
- 64位处理器
- 至少 512MB 可用内存

请查看 README.md 文件获取完整的使用指南和功能说明。
